# 电影推荐系统 python722

## 项目概述

本电影推荐系统基于协同过滤算法，使用 Python 语言开发，结合 Django 框架进行 Web 层实现，使用 SQLite 作为数据库管理系统。

## 技术栈

- **后端框架**：Django
- **编程语言**：Python
- **数据库**：SQLite

## 功能模块

- 用户管理
  - 用户注册与登录
  - 用户偏好设置
- 电影信息管理
  - 电影数据展示
  - 电影分类与搜索
- 推荐算法
  - 基于用户的协同过滤
  - 基于物品的协同过滤
- 推荐结果展示

## 系统角色

- **普通用户**：浏览电影、设置偏好、接收电影推荐
- **系统管理员**：管理电影数据、监控系统运行状态

## 运行环境

- Python 3.8 或以上版本
- SQLite 3.34 或以上版本
- 适用于主流操作系统（Windows、Linux、macOS）

## 部署步骤

1. 确保系统环境满足要求
2. 克隆或下载源代码
3. 安装依赖库（使用 pip）
4. 配置数据库（SQLite）
5. 运行 Django 应用
